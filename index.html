Food diary

Food diary application for tracking macronutrient and energy consumption. This is my latest solo project and the first venture into the wonderful world of Node.js development. My goal was to create a free, easy way to track macronutrient and energy consumption and provide comprehensive coverage of finnish food products and ingredients by combining Fineli's database with web scraped content.

Techical Details

  • Node.js / MySQL back-end (exposes resources via RESTful API).
  • UI written in React (consumes the API).
  • Client-side routing with React Router for SPA experience.
  • Responsiveness with Bootstrap 3 and custom CSS.
  • Web scraping algorithm with Node.js and Cheerio.

Keywords

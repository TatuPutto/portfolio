Food diary

Food diary application for tracking macronutrient and energy consumption. This is my latest solo project and the first venture into the wonderful world of Node.js development. My goal was to create a free, easy way to track macronutrient and energy consumption and provide comprehensive coverage of finnish food products and ingredients by combining Fineli's database with web scraped content.

Techical Details

  • Node.js / MySQL back-end (exposes resources via RESTful API).
  • UI written in React (consumes the API).
  • Client-side routing with React Router for SPA experience.
  • Responsiveness with Bootstrap 3 and custom CSS.
  • Web scraping algorithm with Node.js and Cheerio.

Keywords

JavaScript React React Router HTML5 CSS3 LESS Bootstrap Webpack Single-page application Responsive design Node.js MySQL REST JSON Web scraping Cheerio

Code Snippet Manager

Web application for managing code snippets (gists). My goal was to create an alternative UI for GitHub Gists that combines managing your own code snippets with the discovery potential of Gists service under one easy to use package. This project also constitutes as the practical part of my thesis for my Bachelor's degree.

Techical Details

  • Java / Spring back-end (authentication).
  • UI written in React (consumes Gist API).
  • Flux architechture implemented on the client-side with Redux.
  • Client-side routing with React Router for SPA experience.
